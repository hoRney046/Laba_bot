Проект представляет собой телеграм-бота с мини-игрой лабиринт, в который можно поиграть
в любое время, выполнен с помощью библиотек telebot, pygame и т.д. Для пользования
необходимо запустить бота с помощью команды /play, команду /skin для просмотра ваших скинов,
команду /lottery для получения скинов, и команду /1 для выбора скина.
